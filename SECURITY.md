# Security Policy

UsefulAllays is a Minecraft server plugin. Please do not report security issues through public GitHub issues if they can be abused on live servers.

## Reporting

Open a private security advisory on GitHub or contact the repository owner directly.

Please include:

- affected plugin version or commit
- Paper/Minecraft version
- reproduction steps
- expected impact
- logs if available

## Scope examples

Security-relevant issues may include:

- ownership bypasses
- permission bypasses
- item duplication
- economy abuse
- crashes caused by player actions
- data loss on reload/restart
